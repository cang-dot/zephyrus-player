# Zephyrus Player v0.9.9-vision

## 新增

- **自定义歌曲过渡功能**：智能混音引擎三级递进过渡策略（等功率 / 节拍对齐 / 频域拼接），含 BPM Worker 离线分析、AudioScheduler 精准调度、硬件评估推荐
- **段落门控机制**：crossfade 触发时自动检测高潮段与歌词行，等待段落结束后再开始过渡（12 秒硬上限保护）
- **署名类歌词绕过**：检测到作词/作曲/制作人/致谢等关键词时，绕过门控直接开始渐变切歌
- **封面过渡动画**：GSAP 驱动的动态模糊（缓入→振荡→缓出）+ 下一首封面覆盖层渐显 swap
- **UI 过渡状态桥接**：crossfade 期间歌名/作者/封面/背景渐变同步切换
- **BPM 预分析**：Web Worker 离线自相关 BPM 检测，为节拍对齐过渡提供数据

## 修复

- 修复浮动覆盖布局模式下的大部分问题（粉色遮罩、歌单跳转、设置滚动、播放栏双状态、样式切换不刷新）
- 修复 crossfade 期间封面/歌名闪回旧歌的问题（transitionStore.end() 调用时机移至 completeTransition 之后）
- 修复 crossfade 取消时 isCrossfading 卡死导致自动切歌失效的问题
- 修复模糊动画瞬间消失的问题（改为 0.5s 缓出）

## 优化

- 优化部分页面的加载性能与交互体验
- 智能混音设置界面加入硬件评估推荐
- 封面取色级联到全局 UI 主题色

## 准备进行移动端适配

- 已完成移动端基础设施调研
- 后续版本将逐步完成全部视图的移动端响应式适配及 Capacitor Android 原生端口
