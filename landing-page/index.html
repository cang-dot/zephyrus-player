Tech Stack

现代技术栈

Electron 40 + Vue 3.5 + TypeScript 5.9，每一层都选用经过验证的技术，确保性能和可维护性。

◇

桌面框架

Electron 40 提供跨平台支持，主进程管理窗口、托盘、全局快捷键与自动更新。单实例锁防止重复启动，GPU 加速渲染确保动画流畅。

◇

前端框架

Vue 3.5 Composition API + TypeScript 5.9。Pinia 3.0 模块化状态管理，Vue Router hash history 路由，Naive UI 2.43 组件库。

◇

音频引擎

Howler.js 2.2 + Web Audio API。BiquadFilter 链实现 10 段均衡器，AnalyserNode 驱动 RMS + 频谱高潮检测，GainNode 音量控制。

◇

动画与视觉

GSAP 3.15 提供 27+ 动画预设与 ScrollTrigger 滚动驱动动画。OGL WebGL 渲染粒子/烟幕效果。封面取色实现 CSS 自定义属性级联主题。

◇

构建与工具链

electron-vite 5 + Vite 6.4 构建。Tailwind CSS 3.4 + SCSS 样式系统。electron-builder 26 打包 Windows NSIS / macOS DMG / Linux AppImage。

◇

国际化与 API

vue-i18n 11.2 支持 5 种语言（简中/繁中/英/日/韩）。netease-cloud-music-api-alger 本地 Express 服务提供音乐 API。@unblockneteasemusic/server 解锁多平台音源。
